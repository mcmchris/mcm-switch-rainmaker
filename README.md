# MCM Switch RainMaker
Smart Switch code for the Xiao ESP32C3 with ESP RainMaker

## Connection pinout

| Xiao Expansion Board Pin  | Peripheral  |
| ------------------------- | -------------- |
| D0              | Button   |
| D7              | Relay and Triac output   |
